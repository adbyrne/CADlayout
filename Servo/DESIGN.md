# TrainOrderServo — Design Notes

## Overview

Servo bracket assembly for the Train Order signal mechanism. Mounts two micro-servos
side-by-side (inline along the X axis) above the layout roadbed. The servo arms drive
the signal arms via Bowden-cable wires that pass through the base mast hole.

## Geometry

All dimensions in mm. Coordinate origin is the mast hole centre.

| Feature | Value |
|---------|-------|
| Base plate thickness | 5 mm |
| Mast hole diameter | Ø6 mm |
| Bracket width (each) | 32 mm |
| Bracket height | 25 mm above base |
| Shelf depth (Y) | 4 mm |
| Leg depth (Y) | 4.4 mm additional |
| Leg width (X) | 4.5 mm each |
| Gap bracket → mast | 5 mm each side |
| Servo pocket diameter | Ø11.9 mm |
| Servo pocket centre Z | 13 mm above base bottom |
| Wire/arm slot diameter | Ø6 mm (overlapping servo pocket) |
| Mount holes | Ø3 mm, Y-axis through shelf |
| Cable holes | Ø2 mm, Y-axis through full depth |
| PCA9685 standoffs | Ø3.5 mm lower post × 3 mm, Ø2.3 mm upper × 3 mm |

## Bracket Y Offset (arm clearance)

**Problem (v1):** Both brackets shared the same Y position, so servo arms swept
through the same plane and could collide.

**Fix (v2):** Bracket 2 is shifted **+4 mm in Y** relative to bracket 1.

```
B1 shelf:  Y = -9 to -5   (arm sweeps at Y ≈ -9 front face)
B2 shelf:  Y = -5 to -1   (arm sweeps at Y ≈ -5 front face)
Clearance: 4 mm gap − 1.5 mm arm thickness = 2.5 mm
```

## Variants

### TrainOrderServoInLine_v2

Full assembly with PCA9685 control board mounting.

| Body | Board position | Base |
|------|---------------|------|
| `Body` | +X side of brackets | 150 × 42 mm |
| `Body_Flipped` | −X side (mirror) | 150 × 42 mm |
| `BodyBoardBehind` | −Y direction (in front of brackets) | 80 × 47 mm |

### TrainOrderServoBracketsOnly_v1

Two staggered brackets only — for installations where the PCA9685 is mounted elsewhere.

- Base: 80 × 27 mm
- Mast hole and wire hole retained

### TrainOrderServoSingleBracket_v1

Single bracket — for single-signal installations.

- Base: 55 × 27 mm
- Mast hole and wire hole retained

## Script

All geometry is generated by `scripts/generate_trainorderservo.py`.
Parameters are at the top of the file — no FreeCAD GUI needed to regenerate.

```bash
# From FreeCAD Python console or MCP bridge:
exec(open("scripts/generate_trainorderservo.py").read())
run("/path/to/Servo")
```
