# PowerBox

3D-printable power distribution box for a model railroad layout module.

## Parts

- **PowerBox-Pad003** — Main box body
- **PowerBox-Pocket003** — Lid / cover

## Quick Start

### Print Settings
| Setting | Value |
|---------|-------|
| Material | PETG |
| Printer | Prusa Core One |
| Supports | None |
